PING

Active Work Dates : Summer 2017
Role(s): Game Designer, Programmer
Languages Used: C# (Unity Engine)

Project Description:

  A small personal project aimed at refamiliarizing myself with Unity for an upcoming class. The concept was a game where the player has to navigate a maze and find the exit. The catch of the game is that the player is 'blind' and has to find the way across the play area without hitting the void or a hazard. Additionally, they have a limited of pings (a way to see what is in the immediate area around them) to navigate. The project was intended for a mobile device.
